Prerequisites

1. Propositional Logic

Logic forms the bedrock of mathematical reasoning. It provides the rules and tools to determine the truth or falsehood of statements and to construct valid arguments.

1.1 Statements and Truth Values

A statement (or proposition) is a declarative sentece that is unequivocally either true (T) or false (F).

Examples of Statements:

  • "4 is an even number." (T)
  • "15 is divisible by 7". (F)
  • "The square root of 2 is irrational." (T)

Not statements:

  • "Let's go!"
  • "What is your name?"
  • "All natural numbers are green."

Because these can't be assigned a truth value.

1.2 Logical Operators

We can comine simple statements using logical connectives to form compound statements. The truth value of a compound statement depends entirely on the truth values of its components and the connective used.

Operator Meaning Example Truth Condition
\(\neg A\) NOT A (Negation) "Not raining" True if A is false
\(A \wedge B\) A AND B (Conjunction) "Raining and cold" True only if both are true
\(A \vee B\) A OR B (Disjunction) "Raining or cold" True if at least one true
\(A \Rightarrow B\) IF A THEN B (Implication) "If raining, street is wet" False only if A true and B false*
\(A \Leftrightarrow B\) IF AND ONLY IF (Equivalence) "It rains exactly when the sky is cloudy" True if both have the same true value

*If A is false, the implication is considered true by convention ("from falsehood, anything follows").

Truth Table:

A B A ∧ B A ∨ B A ⇒ B A ⇔ B
T T T T T T
T F F T F F
F T F T T F
F F F F T T

1.3 Tautologies, Paradoxes and Equivalence

Some compound statements are special because their truth value is independent of their components.
They are always true or always false, no matter what.

  • A Tautology: always true.
    Example: \(¬A \vee A\) (The Law of the Excluded Middle)
  • A Paradox: always false.
    Example: \(¬A \wedge A\)

Two statements are logically equivalent (\(A \Leftrightarrow B\)) if they have identical truth tables.
If \(A\) and \(B\) are equivalent, then \(A \Leftrightarrow B\) is a tautology.

1.4 Fundamental Logical Rules

Rules that let you simplify or manipulate logical expressions:

  • Commutativity: \(A \wedge B \Leftrightarrow B \wedge A\) and \(A \vee B \Leftrightarrow B \vee A\)
  • Associativity: \((A \wedge B) \wedge C \Leftrightarrow A \wedge (B \wedge C)\) and \((A \vee B) \vee C \Leftrightarrow A \vee (B \vee C)\)
  • Distributivity: \(A \wedge (B \vee C) \Leftrightarrow (A \wedge B) \vee (A \wedge C)\) and \(A \vee (B \wedge C) \Leftrightarrow (A \vee B) \wedge (A \vee C)\)
  • De Morgan's Laws:
    • \(\neg(A \wedge B) \Leftrightarrow \neg A \vee \neg B\)
    • \(\neg(A \vee B) \Leftrightarrow \neg A \wedge \neg B\)
  • Double Negation: \(\neg(\neg A) \Leftrightarrow A\)
  • Implication rule: \((A \Rightarrow B) \Leftrightarrow (\neg A \vee B)\)
  • Contraposition: (\(A \Rightarrow B) \Leftrightarrow (\neg B \Rightarrow \neg A)\)

1.5 Proof and Implications Rules

  • Equivalence of Implication and Contrapositive:
    \((A \Rightarrow B) \Leftrightarrow (\neg B \Rightarrow \neg A)\)
    "If it rains, the street is wet" \(\Leftrightarrow\) "If the street is not wet, it did not rain".
    Proving the contrapositive is often an easier way to prove an original statement.
  • Equivalence and Implication:
    \((A \Rightarrow B) \Rightarrow (A \Rightarrow B) \wedge (B \Rightarrow A)\)
    This is a vital proof strategy.
  • Transitivity:
    \((A \Rightarrow B) \wedge (B \Rightarrow C) \Rightarrow (A \Rightarrow C)\) and \( (A \Leftrightarrow B) \wedge (B \Leftrightarrow C) \Rightarrow (A \Leftrightarrow C)\)
    This allows to build complex proofs through a chain of simpler implications
  • Proof by Contradiction:
    To prove \(A ⇒ B\) assume \(A ∧ ¬B\) and show this leads to a contradiction.
    This works because \((A ⇒ B) ⇔ ¬(A ∧ ¬B)\)

1.6 Predicate Logic and Quantifiers

Propositional logic deals with whole statements.
Predicate logic introcudes more granularity with variable and quantifiers.

Quantifiers:

Symbol Meaning Example
\(\forall\) "for all" (The Universal Quantifier) \(\forall x, x^2 ≥ 0\)
\(\exists\) "there exists" (The Existential Quantifier) \(\exists x: x^2 = 4\)
\(\exists!\) "there exists exactly one" \(\exists! x: x+2=5\)

Negation Rules for Quantifiers:

  • \(\neg(\forall x: P(x)) \Leftrightarrow \exists x: \neg P(x)\)
  • \(\neg(\exists x: P(x)) \Leftrightarrow \forall x: \neg P(x)\)

Order matters:

  • \(\forall k \in \mathbb{Z} \exists l \in \mathbb{Z}: l ≤ k\) is true
    (for every integer, there's a smaller or equal one)
  • \(\exists l \in \mathbb{Z} \forall k \in \mathbb{Z}: l ≤ k\) is false
    (there's no single integer ≤ every integer)

Quantifiers over Sets:

  • \(\forall x \in \mathbb{M}: P(x) \Leftrightarrow \forall x: (x \in \mathbb{M} \Rightarrow P(x))\)
  • \(\exists x \in \mathbb{M}: P(x) \Leftrightarrow \exists x: (x \in \mathbb{M} \wedge P(x))\)

2. Set Theory

2.1 Basic Concepts

A set is a collection of distinct objects (called elements).

Notation: \(x \in \mathbb{M}\) means "x is an element of M"

Important Sets:

  • \(\emptyset\) or \(\{\}\): the empty set
  • \(\mathbb{N} = \{1, 2, 3, ...\}\): natural numbers
  • \(\mathbb{N}_0 = \{0, 1, 2, 3, ...\}\): natural numbers with zero
  • \(\mathbb{Z} = \{..., -2, -1, 0, 1, 2, ...\}\): integers
  • \(\mathbb{Q} = \{p/q: p \in \mathbb{Z} \wedge q \in \mathbb{Z} \backslash \{\emptyset \}\}\): rational numbers
  • \(\mathbb{R}\): real numbers
  • \(\mathbb{C} = \{x + iy: x, y ∈ ℝ\}\): complex numbers

Russell's Paradox:

\(R = \{ x | x \notin x\} \) leads to the contradiction \(R \in R \Leftrightarrow R \notin R\)
This motivated the development of axiomatic set theory.

2.2 Set Operations and Relations

Operation Symbol Meaning
Subset \(A \subseteq B\) Every element of \(A\) is in \(B\)
Proper Subset \(A \subset B\) \(A \subseteq B\) but \(A \ne B\)
Equality \(A = B\) A equals B
Union \(A \cup B\) Elements in \(A\) or \(B\)
Intersection \(A \cap B\) Elements in both
Difference \(A \backslash B\) Elements in \(A\) not in \(B\)
Complement \(A^c\) Everything not in \(A\) (within a universe)*
Power set \(\mathcal{P}A\) All subset of \(A\)
Cartesian product \(A \times B\) All ordered pairs \((a,b)\)

*If \(A \subseteq \mathbb{U}\) (universe), then \(A^c = \mathbb{U} \backslash A\)

For families of sets \((A_i)_{i \in I}\) with index set I:

  • Union:
    \(\bigcup_{i \in I}\:A_i = \{x: \exists i \in I: x \in A_i\}\) := "somewhere in at least one"
  • Intersection:
    \(\bigcap_{i \in I} \: A_i = \{x:\forall i \in I: x \in A_i\}\) := "in all of them"

2.3 Key Rules

  • De Morgan's Laws:
    \((A \cap B)^c = A^c \cup B^c\) and \((A \cup B)^c = A^c \cap B^c\)
  • Distributive Laws:
    \(A \cap (B \cup C) = (A \cap B) \cup (A \cap B)\) and \(A \cup (B \cap C) = (A \cup B) \cap (A \cup B)\)
  • Inclusion-Exclusion Principle:
    For finite sets: \(|A \cup B\ | = |A| + |B| - |A \cap B|\)
  • Power Set Cardinality:
    For finite A: \(\mathcal{P}(A) = 2^{\{|A|\}}\)

3. Relations and Functions

3.1 Relations

A relation \(R \subseteq X \times Y\) links elements of \(X\) to \(Y\).
Write \(xRy\) for "\((x,y) \in R\)".

Properties of Relations on a Set X:

  • Reflexive: \(\forall x \in X: xRx\)
  • Symmetric: \(\forall x,y \in X: xRy \Rightarrow yRx\)
  • Antisymmetric: \(\forall x,y \in X: (xRy \wedge yRx) \implies x = y\)
  • Transitive: \(\forall x,y,z \in X: (xRy \wedge yRz) \implies xRz\)

Inverse Relation: \(R^{-1} = \{(y,x): (x,y) \in R\}\)

3.2 Equivalence Relations

An equivalence relation is reflexive, symmetric and transitive.
If \(R\) is an equivalence relation, then one often writes \(x \sim y\) instead of \(xRy\).

Examples:

  • Equality (=):
    Define \(xRy \Leftrightarrow x = y\).
    Reflexive: \(x = x\)
    Symmetric: \(x = y \Rightarrow y = x\)
    Transitive: \(x = y \wedge y = z \Rightarrow x = z\)
  • Congruence modulo m: \(x \equiv y \pmod{m} \Leftrightarrow m | (x - y)\)
    Let \(m \in \mathbb{N}\) and \(x,y \in \mathbb{Z}\)
    Define \(xRy :\Leftrightarrow m | (x-y)\)
    Then:
    Reflexive: \(m | (x-x) = 0\)
    Symmetric: \(m | (x-y) \Rightarrow m | (y-x)\)
    Transitive: \(m | (x-y)\) and \(m | (y-z) \Rightarrow m | (x-z)\)
  • Equivalence by decomposition:
    Suppose \(A = \dot{\bigcup}_{i \in I}A_i\) is a disjoint union of nonempty subsets \(A_i\).
    Define \(xRy \Leftrightarrow \exists i \in I \text{ such that } x,y \in A_i\) to the same subset \(A_i\) for some \(i \in I\)
    Reflexive: Every \(x \in A \) lies in some \(A_i\), so \(x,x \in A_i \Rightarrow xRx \)
    Symmetric: If \(xRy\), then \(x,y \in A_i\) for some i; hence also \(y,x \in A_i \Rightarrow yRx\)
    Transitive: If \(xRy\) and \(yRz\), then \(x,y \in A_i\) and \(y,z \in A_j\). Since the union is disjoint
    and \(y \in A_i \cap A_j\), we get \(i = j\), so \(x,z \in A_i \Rightarrow xRz\)
    Thus, \(R\) is an equivalence relation on \(A\).

Equivalence Classes and Quotient Set:

The equivalence class of an Element \(a \in X\) under \(R\) is
\([a] = \{x \in X : xRa \}\), the set of all elements equivalent to \(a\).
So, \([a]\) contains all elements that are “equivalent” to \(a\) under the relation \(R\).

Two key properties hold:

  1. Equality of Classes:
    \([a] = [b]\) if and only if \(aRb\)
    This means:
    • If \(a\) and \(b\) are equivalent (related), then their equivalence classes are the same.
    • Conversely, if two equivalence classes are equal, their representatives are related.
  2. Disjointness of Classes: \([a] \cap [b] = \emptyset\) if \(a\:R\not{b}\)
    Meaning:
    If \(a\) and \(b\) are not related, then their classes have no elements in common.

The set of all distinct equivalence classes of \(X\) under \(R\) is called the quotient set (or partition), denoted:
\(X\backslash R := \{[x]:x \in X\}\)
This partitions \(X\) into non-overlapping subsets (the equivalence classes),
where each element of \(X\) belongs to exactly one class.
It satisfies \(X = \dot{\bigcup}_{[x] \in X/R}[x] \)

Example:

Let \(X = \{1,2,3,4,5,6\}\) and define \(aRb\) if \(a\) and \(b\) have the same remainer when divided by 3.

Then:

  • \([1]=\{1,4\}\)
  • \([2]=\{2,5\}\)
  • \([3]=\{3,6\}\)

And the quotient set is:
\(X/R=\{\{1,4\},\{2,5\},\{3,6\}\}\)

3.3 Order Relations

An order relation organizes elements by comparison.

A partial order is a relation that is:

  • reflexive
  • antisymmetric
  • transitive

Examples:

  • \(≤\) on \(\mathbb{R}\) (total order)
  • \(\subseteq\) on \(\mathcal{P}(A)\) (partial order)
  • Lexicographic (dictionary) order

A total order is a partial order where every two elements are comparable.
\(\forall a,b\) either \(aRb\) or \(bRa\)

So:

  • Partial order = some elements are comparable
  • Total order = every pair is comparable

3.4 Functions

A function \(\mathcal{f} : X \rightarrow Y\) is a relation where each \(x \in X\) is related to exactly one \(y \in Y\)

Notation: \(y = \mathcal{f}(x)\) where \(y\) is the image of \(x\), and \(x\) is a preimage of \(y\).

For \(A \subseteq X\), \(B \subseteq Y\):

  • Image: \(\mathcal{f}(A) = \{\mathcal{f}(x): x \in A\}\)
  • Preimage: \(\mathcal{f}^{-1}(B) = \{x \in X: \mathcal{f}(x_1) = \mathcal{f}(x_2) \Rightarrow x_1 = x_2\}\)

Properties:

  • Injective (one-to-one): \(\forall x_1 x_2 \in X: \mathcal{f} (x_1) = \mathcal{f} (x_2)\Rightarrow x_1 = x_2\)
  • Surjective (onto): \(\forall{y} \in{Y} \exists{x} \in{X} : \mathcal{f}(x) ={y}\)
  • Bijective: Both injective and surjective

Special Functions:

  • Identity: \(\mathcal{id}_x : X \rightarrow X\), \(\mathcal{id}_x (x) = x\)
  • Inclusion: \(\mathcal{i}_a : A \rightarrow X\), \(\mathcal{i}_a (x) = x\) for \(A \subseteq X\)
  • Restriction: \(\mathcal{f} |_a : A \rightarrow Y\), \(\mathcal{f} |_a (x) = \mathcal{f} (x) \) for \(A \subseteq X\)
