Introducing C

C is a general-purpose programming language that has been around for years.

It has been used to write everything from operating systems (including Windows and many others)
to complex programs like the Python interpreter, Git, Oracle database, and more.

1. Output

Most computer programs are designed to produce outputs.

Examples:

  • "You've got a new message" notifications
  • "Game over" displayed on the screen when playing video games
  • Your account balance when checking your online banking app

The simplest output consists of displaying a message on the screen.

#include <stdio.h> int main(){ printf("Hello World!"); return 0; } Hello World!

The printf function is used to display output on the screen. The content to be displayed must be placed inside parentheses.

Displaying Text

When displaying text, it must be enclosed in double quotation marks " ".
In programming, text enclosed in quotation marks is referred to as a string.

2. Statements

A line of code is called a statement. A statement performs a specific task.
The output command is an example of a statement.

Each statement in C ends with a semicolon ;.
Without a semicolon, the program will not run and produce an error.

3. Program Strucure

Header Files

The first line of the code adds the standard input-output library to the program.

#include <stdio.h>

A header file contains definitions of functions and macros.
The stdio.h header file contains the definition of the printf function.
This is why we need to include it at the beginning of our program.

The #include command is used to add a header file to the program.

Main Function

The main() function is the starting point of every C program.
Curly brackets "{ }" indicate the beginning and the end of a function, which can also be called a function's body.

int main(){ printf("Hello World!"); return 0; }

When you run the code, printf is executed, because it is inside the main() function.

Return Statement

The return 0; statement indicates that the program ended successfully.

Example:

#include <stdio.h> int main(){ printf("First"); printf("Second"); return 0; } FirstSecond

This output is happening because the printf command does not add a new line at the end of the output by default.

4. Output Formatting

In all previous examples, we have output strings.
A lot of programs work with numbers. A game for example might need to display the player's score.
If you try to output a number without quotation marks, like this:

#include <stdio.h> int main(){ printf(420); return 0; } Error: type mismatch in printf

You get an error, because the printf function expects a string as input.
To output numbers, we need to use format specifiers, which allow to output numbers and output formatting:

#include <stdio.h> int main(){ printf("Score: %d", 420); return 0; } Score: 420

Here, %d is a format specifier that tells the printf function to expect an integer number as input.
The number 420 is then inserted at the position of the format specifier in the output string.

Common Format Specifiers:

  • %d: Integer number (e.g. 42)
  • %f: Floating-point number (e.g. 3.14)
  • %c: Single character (e.g. 'A')
  • %s: String of characters (e.g. "Hello")
#include <stdio.h> int main(){ printf("Integer: %d\n", 42); printf("Float: %f\n", 3.14); printf("Character: %c\n", 'A'); printf("String: %s\n", "Hello"); return 0; } Integer: 42 Float: 3.140000 Character: A String: Hello

Escape Sequences

In the example above, we used the \n escape sequence to add a new line after each output.

Common Escape Sequences:

  • \n: New line
  • \t: Tab space
  • \\: Backslash
  • \": Double quotation mark
#include <stdio.h> int main(){ printf("Hello,\n\tWorld!"); return 0; } Hello, World!

5. Variables

A variable lets you store a value (like a number or a string) by giving it a name.
You can then use this name to refer to the value later in your code.

To create a variable, you need to declare it.
A variable declaration consists of a data type and a name.

#include <stdio.h> int main(){ int score; score = 111; printf("Score: %d", score); return 0; } Score: 111

After declaring the variable score of type int (integer), we assign it the value 100 using the assignment operator =.
We can then use the variable name score in the printf function to output its value.

Declaration and assignment can also be done in a single line:

#include <stdio.h> int main(){ int score = 222; printf("Score: %d", score); return 0; } Score: 222

A variable's value can be changed at any time:

#include <stdio.h> int main(){ int score = 0; score = 50; printf("Score: %d", score); return 0; } Score: 50

6. Data Types

Variables need a data type that defines what kind of value they can store.
Common data types in C include:

  • int: Integer numbers (e.g. 42, -7)
  • float: Floating-point numbers (e.g. 3.14, -0.001)
  • char: Single characters (e.g. 'A', 'z')
  • double: Double-precision floating-point numbers (e.g. 3.14159)

float vs double

float uses less storage in the memory, but is not as precise as the double type.
So, calculations with float variables are faster than the ones that use double, however, they are less accurate.

#include <stdio.h> int main(){ float pi_float = 3.14f; double pi_double = 3.141592653589793; printf("Float: %f\n", pi_float); printf("Double: %lf\n", pi_double); return 0; } Float: 3.140000 Double: 3.141593

Generally, float is sufficient for storing 7 decimal digits, while double can store up to 15 decimal digits. To declare a float variable, it's common to add the suffix f to the number.
When using the printf function, use %f for float and %lf for double.
Decimal lengths can be controlled by specifying the number of digits after the decimal point, like this: %.2f

#include <stdio.h> int main(){ float pi = 3.14159f; printf("Pi (2 decimals): %.2f\n", pi); printf("Pi (4 decimals): %.4f\n", pi); return 0; } Pi (2 decimals): 3.14 Pi (4 decimals): 3.1416

Char

#include <stdio.h> int main(){ char grade = 'A'; printf("Grade: %c", grade); return 0; } Grade: A

The char data type is used to store a single character enclosed in single quotation marks ' '.

Constants

Variables can be defines as constants. Constants are values that cannot be changed.
To define a constant, use the const keyword:

#include <stdio.h> int main(){ const float PI = 3.14159; printf("Value of PI: %f", PI); return 0; } Value of PI: 3.141590

Here, the variable PI is defined as a constant of type float.
Any attempt to change its value later in the code will result in an error.

Constants must be initialized at the time of declaration.

#include <stdio.h> int main(){ const float PI; PI = 3.14159; printf("Value of PI: %f", PI); return 0; } Error: assignment of read-only variable 'PI'

7. Doing Math

C supports basic arithmetic operations like addition, subtraction, multiplication, and division.

#include <stdio.h> int main(){ int a = 10; int b = 5; printf("Addition: %d\n", a + b); printf("Subtraction: %d\n", a - b); printf("Multiplication: %d\n", a * b); printf("Division: %d\n", a / b); return 0; } Addition: 15 Subtraction: 5 Multiplication: 50 Division: 2

The operators for basic arithmetic are:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division

In case of division, if both operands are integers, the result will also be an integer (the decimal part is truncated).

#include <stdio.h> int main(){ int kb = 35800; int mb = kb / 1024; printf("Megabytes: %d", mb); return 0; } Megabytes: 34

To get a more precise result when dividing, at least one of the operands must be a floating-point number:

#include <stdio.h> int main(){ int kb = 35800; float mb = kb / 1024.0f; printf("Megabytes: %f", mb); return 0; } Megabytes: 34.960938

Remainder

The modulus operator % is used to get the remainder of a division:

#include <stdio.h> int main(){ int items = 100; int per_box = 8; int leftover = items % per_box; printf("Leftover items: %d", leftover); return 0; } Leftover items: 4

8. Comments

Comments are used to add notes or explanations within the code.
They are ignored by the compiler and do not affect the program's execution.

In C, there are two types of comments:

  • Single-line comments start with // and continue to the end of the line.
  • Multi-line comments are enclosed between /* and */.

9. Taking Input

The scanf function is used to take input from the user based on the given format specifier.

#include <stdio.h> int main(){ int age; printf("Enter your age: "); scanf("%d", &age); printf("You are %d years old.", age); return 0; }

In the outputs, I'll be using example inputs.

Enter your age: 25 You are 25 years old.

The scanf function reads the input provided by the user and stores it in the variable specified.
The ampersand symbol & is used to provide the memory address of the variable where the input value should be stored. More on that later.

Multiple Inputs

You can take multiple inputs in a single scanf call: #include <stdio.h> int main(){ int a, b; printf("Enter two numbers: "); scanf("%d %d", &a, &b); printf("Sum: %d", a + b); return 0; } Enter two numbers: 7 3 Sum: 10

Notice how we were able to declare two variables on a single line, seperated by commas, because they are of the same type.

10. Conditionals

Conditionals allow you to execute certain parts of code based on whether a specific condition is true or false.

Decision Making

The if statement allows you to execute a block of code if a condition is true.

if(confition){ // code to run if condition is true }

You can also use the else statement to execute a block of code if the condition is false.

#include <stdio.h> int main(){ int age; printf("Enter your age: "); scanf("%d", &age); if(age >= 18){ printf("You are an adult."); } else { printf("You are a minor."); } return 0; } Enter your age: 20 You are an adult.

The following comparison operators may be used to form the condition:

  • <: less than
  • <=: less than or equal to
  • >: greater than
  • >=: greater than or equal to
  • ==: equal to
  • !=: not equal to

Note that the assignment operator = is different from the equality operator == used in conditions.

In case you have multiple conditions to check, you can use the else if statement:

#include <stdio.h> int main(){ int score; printf("Enter your score: "); scanf("%d", &score); if(score >= 90){ printf("Grade: A"); } else if(score >= 80){ printf("Grade: B"); } else if(score >= 70){ printf("Grade: C"); } else { printf("Grade: F"); } return 0; } Enter your score: 85 Grade: B
