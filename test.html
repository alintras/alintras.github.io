3.2 Equivalence Relations

A relation \(R\) on a set \(X\) is called an equivalence relation if it is reflexive, symmetric, and transitive:

  • Reflexive: \(\forall x \in X: xRx\)
  • Symmetric: \(\forall x,y \in X: (xRy \implies yRx)\)
  • Transitive: \(\forall x,y,z \in X: (xRy \wedge yRz) \implies xRz\)

An equivalence relation can be viewed as a weakened form of equality. In particular, equality itself is an equivalence relation.

Examples:

  1. Equality (=):
    For all \(x,y,z \in X\):
    Reflexive: \(x = x\);
    Symmetric: \(x = y \implies y = x\);
    Transitive: \(x = y \wedge y = z \implies x = z\).
  2. Congruence modulo m:
    Let \(m \in \mathbb{N}\) and \(x,y \in \mathbb{Z}\). Define
    \(xRy :\!\!\iff m \mid (x - y)\).
    Then:
    Reflexive: \(m \mid (x - x) = 0\);
    Symmetric: \(m \mid (x - y) \implies m \mid (y - x)\);
    Transitive: \(m \mid (x - y)\) and \(m \mid (y - z)\) \(\implies m \mid (x - z)\).
    This relation is written \(x \equiv y \pmod{m}\) and is called congruence modulo m.
  3. Equivalence by decomposition:
    Suppose \(A = \dot{\bigcup}_{i \in I} A_i\) is a disjoint union of nonempty subsets.
    Define \(xRy :\!\!\iff \exists i \in I : x \in A_i \wedge y \in A_i.\)
    Then \(R\) is an equivalence relation on \(A\).

Equivalence Classes and Quotient Set:

Given an equivalence relation \(R\) on \(X\), the equivalence class of an element \(a \in X\) is \[ [a] := \{x \in X : xRa\}. \] Two key properties hold:

  • \([a] = [b]\) if and only if \(aRb\)
  • \([a] \cap [b] = \emptyset\) if \(a \not R b\)

The set of all equivalence classes \[ X / R := \{ [x] : x \in X \} \] is called the quotient set (or partition) of \(X\) by \(R\). It satisfies \[ X = \dot{\bigcup}_{[x] \in X/R} [x]. \]
