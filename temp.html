An Introduction to Propositional Logic

Logic forms the bedrock of mathematical reasoning. It provides the rules and tools to determine the truth or falsehood of statements and to construct valid arguments. This article explores the fundamentals of propositional logic.

1. Statements and Truth Values

At its core, propositional logic deals with statements (or propositions). A statement is a declarative sentece that is unequivocally either true (T) or false (F).

Examples of Statements:

  • A: "4 is an even number." (T)
  • B: "15 is divisible by 7". (F)
  • C: "The square root of 2 is irrational." (T)

Sentences like "Let's go!" or "What is your name?" are not statements because they cannot be assigned a truth value.

2. Logical Operators

Simple statements can be combined using logical operators or connectives to form compound statements. The truth value of a compound statement depends entirely on the truth values of its components and the connective used. This is systematically analyzed using truth tables.

The primary operators are:

  • ¬ (NOT): Negation. It reverses the truth value.
  • ∧ (AND): Conjunction. A∧B is true only if both A and B are true.
  • ∨ (OR): Disjunction. A∨B is true if at least one of A or B is true. This is the "inclusive or".
  • ⇒ (IMPLIES): A⇒B means "if A, then B".
    It is false only when A is true and B is false.
    If A is false, the implication is considered true by convention ("from falsehood, anything follows").
  • ⇔ (IF AND ONLY IF): Equivalence. A⇔B is true when A and B have the same truth value.
